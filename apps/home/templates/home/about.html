{% extends "base.html" %} {% block title %}About{% endblock %} {% block header %}About this site{% endblock %} {% block content %} Footbag freestyle is a technically challenging sport. Being able to find quality information when you need it is therefore very important. By having a comprehensive collection of move demonstration videos and tutorials along with trick tips we are aiming to reduce the learning curve as much as possible.

Until now if you didn't know the names of various moves it was hard to find any information. With the development of the footbag DB you have better ways to find relevant footbag information. This project extends beyond just organizing pre-existing information. By leveraging the information there is the potential to do new things to help people learn more about the spot.

The code powering this site can be found over on the github project page. This is open source code so do check it out if you are technially inclined, bug reports and patches are welcome!

This site is intended to be an educational resource much like wikipedia. If you are looking for other footbag related information like nearby clubs and upcoming events you could visit: footbag.org If you are looking for a "social" site you could visit: modified footbag forum {% endblock %}
